## WOOYANG CRM / ERP

사내 영업·문서·재고·생산·결재를 통합 운영하기 위한 Next.js + Supabase 기반 ERP 프로젝트입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 주요 기능 도메인

- CRM/상담 관리
- 문서 관리(견적/발주/의뢰)
- 재고/생산 관리
- 결재/게시판/알림
- 관리자 권한/메뉴 정책

## 이번 개선(ERP 지향)

- TODO API 인증 강화를 통해 사용자 본인 데이터만 접근하도록 개선.
- 대시보드에 `ERP 업무 흐름 보드` 추가:
  - 상담 → 문서 → 생산/재고 → 실행 → 출고/배송 단계별 현황 확인.
  - 각 단계에서 관련 메뉴로 즉시 이동 가능.

## 참고

환경 변수 및 Supabase 스키마는 사내 운영 정책에 맞게 관리해야 합니다.
